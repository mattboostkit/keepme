Favicon Test Page

This page tests all the favicon files in this directory.

favicon.ico

This is the classic favicon file that should contain multiple sizes (16x16, 32x32).

HTML: <link rel="shortcut icon" href="favicon.ico">

favicon.svg

This is a scalable vector version of the favicon.

HTML: <link rel="icon" type="image/svg+xml" href="favicon.svg">

favicon-96x96.png

This is a 96x96 pixel PNG version of the favicon.

HTML: <link rel="icon" type="image/png" sizes="96x96" href="favicon-96x96.png">

apple-touch-icon.png

This is used for iOS devices when adding the site to the home screen.

HTML: <link rel="apple-touch-icon" href="apple-touch-icon.png">

web-app-manifest-192x192.png

This is used for Android devices and PWAs.

Referenced in: site.webmanifest

web-app-manifest-512x512.png

This is a larger version used for Android devices and PWAs.

Referenced in: site.webmanifest
